    Transform

    A Transform is a class that represents a position and a size.

    # new Transform(kwargs)

    Parameters:
    Name Type Description
    kwargs Object

    The keyword arguments.

    Properties
    Name Type Attributes Description
    pos Vector2 <optional>

    The world position. (Optional with x and y properties)
    size Vector2 <optional>

    The world size. (Optional with w and h properties)
    rot Number

    The world rotation.
    x Number

    The x position.
    y Number

    The y position.
    w Number

    The width.
    h Number

    The height.
    Source:
    • maths/transform.js, line 4
    Example 
    // Create a new transform.
let transform = new Transform({
     pos: new Vector2(10, 10),
     size: new Vector2(10, 10),
     rot: 0
   });

    # align(type)

    Aligns the transform to the parent.

    Available Types Description
    center Aligns the transform to the center of the parent.
    top Aligns the transform to the top of the parent.
    bottom Aligns the transform to the bottom of the parent.
    left Aligns the transform to the left of the parent.
    right Aligns the transform to the right of the parent.
    middle Aligns the transform to the middle of the parent.
    Parameters:
    Name Type Description
    type String

    The type of alignment.
