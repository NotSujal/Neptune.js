    Input

    Input

    The Input class is capable of handling keyboard, mouse and touch inputs.

    Constructor

    # new Input()

    Properties
    Name Type Description
    canvas HTMLCanvasElement

    The canvas element that the input is attached to.
    pos Object

    The position of the mouse or touch.

    Properties
    Name Type Description
    x number

    The x position of the mouse or touch.
    y number

    The y position of the mouse or touch.
    pressedButton Number

    The button that is currently pressed.
    isClicked Boolean

    Returns true if the button is clicked.
    isDoubleClicked Boolean

    Returns true if the button is double clicked.
    isMouseDown Boolean

    Returns true if the mouse button is down.
    touchCount Number

    Returns the number of touches.
    keyPressed Number

    The key that is currently pressed. (UNICODE)
    specialKeyPressed Number

    The special key that is currently pressed.
    wheelDelta Number

    The amount of wheel movement.
    onClick function

    The function to call when the button is clicked.
    onDoubleClick function

    The function to call when the button is double clicked.
