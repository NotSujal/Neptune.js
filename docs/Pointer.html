    Pointer

    Pointer

    A class that represents a Pointer event. Poniter class is used to detect mouse and touch events on the canvas.

    Constructor

    # new Pointer()

    Creates a new Pointer event.

    Properties
    Name Type Description
    x Number

    The x coordinate of the pointer.
    y Number

    The y coordinate of the pointer.
    dx Number

    The change in x coordinate of the pointer.
    dy Number

    The change in y coordinate of the pointer.
    BUTTON_CODE Object

    The button code.

    Properties
    Name Type Description
    LEFT Number

    The left pointer button code.
    MIDDLE Number

    The middle pointer button code.
    RIGHT Number

    The right pointer button code.
    WHEEL_CODE Object

    The wheel code.

    Properties
    Name Type Description
    UP Number

    The up wheel code.
    DOWN Number

    The down wheel code.
